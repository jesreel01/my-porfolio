### Prerequisites
- Node.js 18+ 
- pnpm, npm, or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/my-porfolio.git
cd my-porfolio
```

2. Install dependencies:
```bash
pnpm install
# or
npm install

yarn install
```

3. Run the development server:

